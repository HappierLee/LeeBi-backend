# 智能Bi

基于 React + Spring Boot + RabbitMQ + AIGC 的智能数据分析平台。

> 用户只需要导入原始数据集、并输入分析诉求，
> 就能自动生成可视化图表及分析结论，实现数据分析的降本增效。
